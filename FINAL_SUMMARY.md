### Две разные системы:

1. **История переводов** (v3.6.0)
   - Хранит готовые файлы
   - В браузере (localStorage)
   - Может теряться
   - Для скачивания старых результатов

2. **Кэш переводов** (v3.3.0) ⭐
   - Хранит отдельные переводы
   - В файле на диске
   - Никогда не теряется
   - Экономит API лимит
