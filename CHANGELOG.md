# Changelog

## [3.1.0] - 2026-05-02

### ✨ Добавлено
- Валидация размера файлов (максимум 1000 MB) с понятными сообщениями об ошибках
- Shared типы для API responses (`AnalyzeResponse`, `TranslateResponse`, `ApiErrorResponse`)
- Улучшенная обработка CDATA и комментариев в XML парсере
- Сохранение whitespace при переводе XML файлов
- Документ `IMPROVEMENTS.md` с рекомендациями по дальнейшему развитию

### 🔧 Исправлено
- Улучшены сообщения об ошибках DeepL API (429, 456, 403, 5xx)
- Исправлена type safety между клиентом и сервером
- Увеличен лимит body size до 1000mb в next.config.js
- XML парсер теперь корректно обрабатывает edge cases

### 🚀 Улучшено
- Более понятные сообщения об ошибках для пользователей
- Лучшая обработка ошибок в API routes
- Улучшенная типизация во всем проекте

### 📝 Технические детали

#### Измененные файлы:
- `lib/deepl.ts` - улучшенная обработка ошибок API
- `lib/langParsers.ts` - исправлен XML парсер
- `app/page.tsx` - добавлена валидация размера файлов и улучшена типизация
- `types/index.ts` - добавлены новые типы для API
- `next.config.js` - увеличен лимит body size

#### Новые файлы:
- `IMPROVEMENTS.md` - рекомендации по улучшению проекта
- `CHANGELOG.md` - история изменений

- CRT-стиль интерфейса
