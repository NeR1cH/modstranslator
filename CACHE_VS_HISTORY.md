## 📊 Две разные системы

### 1️⃣ История переводов (v3.6.0)

**Что это:**
- Сохраняет **готовые файлы** (JAR, SNBT и т.д.)
- Можно скачать результаты предыдущих переводов
- Хранится в **браузере** (localStorage)

**Где хранится:**
```
localStorage браузера
(может очищаться при очистке данных браузера)
```

**Для чего:**
```
✅ Скачать файл который переводил вчера
✅ Посмотреть историю работы
✅ Не переводить заново если случайно закрыл вкладку
```

**Проблема:**
```
❌ Теряется при очистке данных браузера
❌ Не экономит API лимит (хранит файлы, не переводы)
❌ Не переживает перезапуск сервера надежно
```

---

### 2️⃣ Кэш переводов (v3.3.0) ⭐ ЭТО ТО ЧТО ТЕБЕ НУЖНО!

**Что это:**
- Сохраняет **отдельные переводы** (строка → перевод)
- Переиспользует их автоматически
- Хранится в **файле на диске**

**Где хранится:**
```
C:\VSCODE PROJECTS\modstranslator\.translation-cache\cache-v1.json
(файл на диске, НЕ теряется НИКОГДА)
```

**Для чего:**
```
✅ Экономит API лимит (не переводит повторно)
✅ Работает между сессиями
✅ Переживает перезапуск сервера
✅ Переживает перезагрузку компьютера
✅ Работает автоматически
```

**Преимущества:**
```
✅ Сохраняется в файл на диске
✅ НЕ теряется при перезагрузке страницы
✅ НЕ теряется при перезапуске сервера
✅ НЕ теряется при очистке браузера
✅ Работает вечно (пока не удалишь файл)
```

## 🔧 Как это работает технически

### При переводе:

```javascript
// 1. Проверка кэша (файл на диске)
const cache = getTranslationCache(); // Загружает из файла
const cached = cache.getMany(texts);  // Ищет в кэше

// 2. Если в кэше - возвращает сразу
if (cached.has("Magic Sword")) {
  return "Волшебный меч"; // 0 символов API! ✅
}

// 3. Если нет в кэше - переводит через API
const translated = await callDeepLAPI("Magic Sword");

// 4. Сохраняет в кэш (файл на диске)
cache.set("Magic Sword", "Волшебный меч");
// Файл .translation-cache/cache-v1.json обновлен ✅
```

### При запуске сервера:

```javascript
// Автоматически загружает кэш из файла
constructor() {
  this.loadFromDisk(); // Читает .translation-cache/cache-v1.json
  console.log('Loaded 2,500 entries from disk'); ✅
}
```

### При сохранении:

```javascript
// Автоматически сохраняет в файл каждые 5 секунд
private scheduleSave() {
  setTimeout(() => {
    fs.writeFileSync(CACHE_FILE, JSON.stringify(entries));
    console.log('Saved to disk'); ✅
  }, 5000);
}
```

## ❓ FAQ

### Q: Кэш теряется при перезагрузке страницы?

**A:** НЕТ! Кэш хранится в файле на диске, не в браузере.

### Q: Кэш теряется при перезапуске сервера?

**A:** НЕТ! Кэш загружается из файла при каждом запуске.

### Q: Кэш теряется при перезагрузке компьютера?

**A:** НЕТ! Это обычный файл на диске.

### Q: Кэш теряется при очистке данных браузера?

**A:** НЕТ! Кэш не связан с браузером.

### Q: Когда кэш может потеряться?

**A:** Только если ты вручную удалишь файл `.translation-cache/cache-v1.json`.
