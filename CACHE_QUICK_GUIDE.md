# 💰 Краткая инструкция: Экономия API лимита

## ✅ Система кэширования УЖЕ РАБОТАЕТ!

С версии **3.3.0** все переводы автоматически сохраняются и переиспользуются.

---

## 🎯 Как это выглядит

### В UI приложения:

```
┌─────────────────────────────────────────────────────┐
│ 💾 КЭШ ПЕРЕВОДОВ                                    │
│ Закэшировано: 1,234 запись(ей)                  [↻] │
│                                                      │
│ ℹ️ Повторные переводы используют кэш вместо API    │
│    (экономия лимита)                          [🗑️]  │
└─────────────────────────────────────────────────────┘
```

### В логах консоли (F12):

```
[deepl] Cache stats: 150 hits, 50 misses
[deepl] Characters to translate (uncached): 450
[cache] Batch lookup: 150/200 hits (75%)
```

**Расшифровка:**
- **150 hits** = 150 строк взято из кэша (0 символов API) ✅
- **50 misses** = 50 строк переведено через API (450 символов) 💰
- **Экономия: 75%!** 🎉

## 🔍 Как проверить что кэш работает

### Способ 1: Посмотри индикатор в UI

Открой приложение → найди секцию **"💾 КЭШ ПЕРЕВОДОВ"**

Если видишь число > 0 → кэш работает! ✅

### Способ 2: Переведи один файл дважды

1. Переведи любой файл (мод, квест, конфиг)
2. Посмотри в логи: `[deepl] Characters to translate: 1000`
3. Переведи **тот же файл** еще раз
4. Посмотри в логи: `[deepl] All translations found in cache, skipping API call`
5. **0 символов использовано!** ✅

### Способ 3: Посмотри файл кэша

```bash
cat ".translation-cache/cache-v1.json"
```

Если файл существует и содержит записи → кэш работает! ✅

## 💡 Советы по максимальной экономии

### 1. Переводи похожие моды вместе

```
✅ ХОРОШО:
Переведи все магические моды подряд
→ много общих слов (magic, spell, wand)
→ высокая экономия!

❌ ПЛОХО:
Переведи магический мод, потом технический, потом магический
→ кэш не используется эффективно
```

### 2. Не очищай кэш без причины

```
✅ ХОРОШО:
Оставь кэш как есть
→ накапливаются переводы
→ экономия растет!

❌ ПЛОХО:
Очищай кэш после каждого мода
→ теряются все переводы
→ экономия = 0%
```

### 3. Переводи обновления модов

```
✅ ХОРОШО:
Мод v1.0 → перевел
Мод v1.1 → 95% из кэша!
Мод v1.2 → 98% из кэша!

Экономия огромная! 🎉
```
