# 🎉 Решение проблемы с пониманием переводов

## Что было сделано

Добавлена система **детальных отчетов о переводе**, которая решает твою проблему с непониманием что именно переводится в квестах и модах.

---

## Проблема (до)

❌ **Квесты:** Непонятно что переводится - описание? название? текст задания?
❌ **Моды:** Не знаешь какие строки были переведены и что они означают
❌ **Результат:** Приходится скачивать файл и открывать его чтобы проверить

---

## Решение (после)

✅ **Детальный отчет** показывает каждую переведенную строку
✅ **Примеры переводов** - видишь оригинал и перевод до скачивания
✅ **Ключи строк** - понятно что это (item.*, block.*, quest.title и т.д.)
✅ **Форматы отчетов** - HTML (красивый) и TXT (для документации)

---

## Как использовать

### 1. Переведи файл как обычно

```
1. Загрузи файл (.jar, .snbt, .json и т.д.)
2. Нажми "ЗАПУСТИТЬ ПЕРЕВОД"
3. Дождись завершения
```

### 2. Открой отчет

После перевода появится новая секция:

```
// 04. ОТЧЕТ О ПЕРЕВОДЕ
▶ 📊 ОТЧЕТ О ПЕРЕВОДЕ                    [⬇ HTML]

Режим: Одиночный файл
Файлов: 1
Строк переведено: 150
```

Нажми на **▶** чтобы развернуть.

### 3. Посмотри примеры

```
📄 quests/chapter1.snbt [SNBT]
   [SNBT] • 5 строк

   [snbt_line_3]
   EN: Complete the first quest
   RU: Завершите первый квест

   [snbt_line_7]  
   EN: Gather 10 wood logs
   RU: Соберите 10 бревен
```

Теперь ты **точно знаешь** что было переведено!

---

## Примеры для разных форматов

### Квесты (SNBT)

```
[snbt_line_2]
EN: Welcome to the adventure
RU: Добро пожаловать в приключение

[snbt_line_5]
EN: Find the hidden treasure  
RU: Найдите спрятанное сокровище
```

**Переводятся:** title, description, text, name, subtitle

### Моды (JAR)

```
[item.examplemod.magic_sword]
EN: Magic Sword
RU: Волшебный меч

[block.examplemod.ore_block]
EN: Mysterious Ore
RU: Таинственная руда
```

**Переводятся:** названия предметов, блоков, сущностей, интерфейса

### Конфиги (TOML)

```
[toml_line_5]
EN: Enable advanced features
RU: Включить расширенные функции

[toml_line_8]
EN: Server welcome message
RU: Приветственное сообщение сервера
```

**Переводятся:** описания настроек, сообщения

---

## Что показывает отчет

Для каждой переведенной строки:

1. **Ключ** - идентификатор (`item.minecraft.stone`, `snbt_line_5`)
2. **Оригинал (EN)** - исходный английский текст
3. **Перевод (RU)** - переведенный русский текст
