# Как запускать dev-menu.bat из любого места

## 🚀 Быстрая установка (рекомендуется)

### Автоматическая установка:
1. Открой PowerShell
2. Выполни:
   ```powershell
   cd "C:\VSCODE PROJECTS\modstranslator\scripts"
   .\add-to-path.ps1
   ```
3. Перезапусти командную строку
4. Готово! Теперь можешь запускать из любой папки:
   ```bash
   dev-menu.bat
   ```

---

## 📝 Ручная установка

### Шаги:
1. Нажми `Win + X` и выбери "Система"
2. Нажми "Дополнительные параметры системы"
3. Нажми "Переменные среды"
4. В разделе "Переменные среды пользователя" найди `Path` и нажми "Изменить"
5. Нажми "Создать" и добавь путь:
   ```
   C:\VSCODE PROJECTS\modstranslator\scripts
   ```
6. Нажми "ОК" везде
7. Перезапусти командную строку

### Использование:
```bash
dev-menu.bat
# или короткий алиас:
d
```

---

## ✅ Проверка

После настройки открой **новую** командную строку и выполни:
```bash
dev-menu.bat
# или просто:
d
```

Должно открыться меню MOD_TRANSLATOR с опциями:
1. Run tests only
2. Start dev server (no tests)
3. Start dev server with test check
4. Start dev server with auto-fix (3 attempts)
5. Exit

---

## 🎯 Другие доступные скрипты

После добавления в PATH также будут доступны:
- `d` - короткий алиас для dev-menu (БЫСТРЫЙ ДОСТУП!)
- `dev-menu.bat` - полное меню разработчика
- `run-tests.bat` - быстрый запуск тестов
- `start-with-tests.bat` - запуск с проверкой
- `start-with-auto-fix.bat` - запуск с авто-исправлением
