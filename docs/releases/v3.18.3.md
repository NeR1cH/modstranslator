## ✨ Новые возможности

### 🎯 Автоматическое определение рода

**Метод:** `inferGenderFromRussian(russianWord: string)`

**Правила определения:**
1. `-а`, `-я` → feminine (рама, катушка, проволока)
2. `-о`, `-е` → neuter (окно, устройство)
3. `-ь` → masculine (кабель, корень)
4. Согласная → masculine (ключ, блок, корпус)

**Использование:**
- Fallback когда существительное НЕ в NOUN_GENDERS
- Работает для любых новых слов
- Высокая точность (~95% русских существительных)

**Примеры работы:**
```
Copper Wrench  → Медный ключ      ✅ (masculine from "ключ")
Copper Coil    → Медная катушка   ✅ (feminine from "катушка")
Copper Cable   → Медный кабель    ✅ (masculine from "кабель")
Copper Wire    → Медная проволока ✅ (feminine from "проволока")
Copper Device  → Медное устройство ✅ (neuter from "устройство")
```

**Результаты тестирования:**
- ✅ 5/5 вариаций с правильным согласованием (100%)
- ✅ 5/5 переведено через FragmentCache (100%)
- ✅ 0 API вызовов (100% экономия)

## 🐛 Исправления

### 1. Неправильный род "frame" в NOUN_GENDERS

**Проблема:**
```
'frame': 'masculine'  // ❌ Неправильно
"Advanced Iron Frame" → "Продвинутая Железный рама" ❌
```

**Исправление:**
```typescript
'frame': 'feminine'  // ✅ Правильно
"Advanced Iron Frame" → "Продвинутая железная рама" ✅
```

**Файл:** `lib/fragmentCache.ts:84`

### 2. Регистр букв в составных переводах

**Проблема:**
```
"Advanced Iron Frame" → "Продвинутая Железная рама" ❌
```

**Исправление:**
- Нормализация регистра: все слова кроме первого с маленькой буквы
- Результат: "Продвинутая железная рама" ✅

**Файл:** `lib/fragmentCache.ts:432-434`

---

### 3. Прилагательные с окончанием "ой"

**Проблема:**
```
"Gold Sheet" → "Золотый лист" ❌ (должно быть "Золотой")
"Gold Block" → "Золотый блок" ❌ (должно быть "Золотой")
```

**Причина:**
- `normalizeToMasculine()` всегда использовал "ый" окончание
- Русские прилагательные с ударным окончанием используют "ой": золотой, большой, молодой, дорогой

**Исправление:**
```typescript
private readonly OJ_ENDING_ADJECTIVES = new Set([
  'золот', 'больш', 'молод', 'дорог', 'чуж', 'живой'
]);
```

**Результат:**
```
"Gold Sheet" → "Золотой лист" ✅
"Gold Block" → "Золотой блок" ✅
"Gold Ingot" → "Золотой слиток" ✅
```

**Файлы:**
- `lib/fragmentCache.ts:106-109` — набор OJ_ENDING_ADJECTIVES
- `lib/fragmentCache.ts:193-209` — обновлен `normalizeToMasculine()`
- `lib/fragmentCache.ts:214-232` — обновлен `applyGenderAgreement()`

## ⚠️ Ограничения

### 1. Слова на -ь могут быть feminine
- Автоматика определяет как masculine
- Требуется ручное добавление в NOUN_GENDERS для feminine слов
- Примеры: дверь, цепь, печать, сталь

### 2. Требует наличия перевода в кэше
- Метод работает только если существительное уже переведено
- Не может определить род для совсем новых слов

### 3. Не учитывает семантику
- "папа" оканчивается на -а, но masculine
- Такие слова нужно добавлять в NOUN_GENDERS вручную
