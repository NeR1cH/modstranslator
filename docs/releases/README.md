## 🎯 Основные вехи

### v3.19.0 - Code Cleanup & Optimization
Удаление неиспользуемой Word-Based системы перевода, которая дублировала функциональность FragmentCache и имела фундаментальные проблемы.

**Ключевые достижения:**
- Удалено 810 строк мёртвого кода
- Упрощена архитектура pipeline (4 шага вместо 6)
- Убрана дублирующая функциональность
- Нет breaking changes
- Все тесты проходят

### v3.18.3 - Automatic Gender Inference
Автоматическое определение рода существительных по окончанию русского перевода с исправлением критических ошибок согласования.

**Ключевые достижения:**
- Автоматическое определение рода (~95% точность)
- Fallback для неизвестных существительных
- Исправлены ошибки согласования (frame, capitalization, "ой" endings)
- Расширен словарь feminine слов на -ь
- 100% покрытие тестами

### v3.18.0 - Universal FragmentCache with Gender Agreement
Полная переработка FragmentCache для работы с любым контентом и автоматическим согласованием по роду в русском языке.

**Ключевые достижения:**
- Универсальное извлечение паттернов (5 стратегий)
- Автоматическое согласование прилагательных с существительными
- Фильтрация stop words
- Убраны лимиты на файлы и запросы
- Экономия памяти (хранится только мужской род)

### v3.17.0 - API Key Rotation System
Система автоматической ротации API ключей DeepL с поддержкой нескольких ключей и автоматическим переключением при исчерпании лимита.

**Ключевые достижения:**
- Поддержка нескольких DeepL API ключей
- Автоматическое переключение при исчерпании
- UI с детальной статистикой
- Обратная совместимость

### v3.16.0 - Word-Based Translation System
Революционное обновление, добавившее интеллектуальную систему пословного перевода с морфологией и автоматическим обучением.

**Ключевые достижения:**
- Пословный перевод с кэшированием
- Морфологическое согласование
- Автоматическое обучение
- Экономия API вызовов

### v3.15.1 - Translation Cache Fix
Критическое исправление кэша переводов, добавившее хранение оригинального текста для возможности извлечения фрагментов.

**Ключевые достижения:**
- Исправлен кэш переводов
- Добавлено извлечение фрагментов
- Улучшена документация

## 📝 Соглашения о версионировании

Проект использует [Semantic Versioning](https://semver.org/):

- **MAJOR** (X.0.0) - Breaking changes, несовместимые изменения API
- **MINOR** (0.X.0) - Новая функциональность, обратно совместимая
- **PATCH** (0.0.X) - Исправления багов, обратно совместимые

### Примеры:
- `v3.16.0` - Minor (новая функциональность: Word-Based System)
- `v3.15.1` - Patch (исправление: Translation Cache)
