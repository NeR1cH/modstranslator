---

## ✨ Новая функциональность

### Ротация API ключей DeepL

**Проблема:**
- Free план DeepL: 500,000 символов/месяц
- При исчерпании лимита перевод останавливается
- Нужно ждать до 1-го числа следующего месяца
- Невозможно использовать несколько Free ключей одновременно

**Решение:**

Реализована система автоматической ротации API ключей с отслеживанием лимита для каждого ключа.

## 💡 Использование

### Один ключ (как раньше):
```env
DEEPL_API_KEY=your-api-key:fx
```

### Несколько ключей (новая функция):
```env
DEEPL_API_KEY=key1:fx,key2:fx,key3:fx
```

### Поведение:
1. Система использует первый ключ
2. При исчерпании лимита (500,000 символов) автоматически переключается на второй
3. При исчерпании второго переключается на третий
4. Если все ключи исчерпаны — показывает ошибку
5. 1-го числа каждого месяца все лимиты сбрасываются

## 📊 Технические детали

**Формат хранения:**
```json
{
  "keys": {
    "78039a4b-f09f-428f-99a8-228e3be191ad:fx": {
      "charactersUsed": 377875,
      "requestsCount": 108,
      "lastReset": "2026-05-03T10:03:06.520Z",
      "monthlyLimit": 500000,
      "status": "exhausted"
    },
    "another-key:fx": {
      "charactersUsed": 0,
      "requestsCount": 0,
      "lastReset": "2026-05-08T19:39:50.206Z",
      "monthlyLimit": 500000,
      "status": "active"
    }
  },
  "currentKeyIndex": 1
}
```

**Логика переключения:**
```typescript
async checkLimit(textLength: number): Promise<void> {
  const remaining = keyStats.monthlyLimit - keyStats.charactersUsed;
  
  if (remaining < textLength) {
    keyStats.status = 'exhausted';
    
    if (this.switchToNextKey()) {
      // Рекурсивно проверяем лимит с новым ключом
      return this.checkLimit(textLength);
    }
    
    throw new Error('Все API ключи DeepL исчерпаны');
  }
}
```
