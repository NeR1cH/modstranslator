## 🛡️ Реализованная защита

### 1. CSRF защита

```typescript
// middleware.ts
if (request.method !== 'GET' && request.method !== 'HEAD') {
  const origin = request.headers.get('origin');
  const host = request.headers.get('host');
  
  if (!origin || !host || !origin.includes(host)) {
    return NextResponse.json({ error: 'CSRF validation failed' }, { status: 403 });
  }
}
```

**Защищает от:**
- Cross-Site Request Forgery атак
- Несанкционированных запросов с других доменов

### 3. Path Traversal защита

```typescript
export function sanitizePath(filePath: string): string {
  const normalized = filePath
    .replace(/\.\./g, '')
    .replace(/^\/+/, '')
    .replace(/\\+/g, '/');
  
  if (!normalized.startsWith('assets/')) {
    throw new Error('Invalid file path: must be inside assets/');
  }
  
  return normalized;
}
```

**Защищает от:**
- Доступа к файлам вне разрешенных директорий
- Атак типа `../../etc/passwd`
- Перезаписи системных файлов

---

### 4. Prototype Pollution защита

```typescript
export function safeJsonParse(text: string): any {
  const obj = JSON.parse(text);
  
  if (obj && typeof obj === 'object') {
    delete obj.__proto__;
    delete obj.constructor;
    delete obj.prototype;
  }
  
  return obj;
}
```

**Защищает от:**
- Модификации прототипов объектов
- Внедрения вредоносного кода через JSON
- Атак на Object.prototype

---

### 5. Валидация размера файлов

```typescript
export function validateBase64Size(base64: string, maxSizeMB: number = 1024): void {
  const sizeBytes = (base64.length * 3) / 4;
  const sizeMB = sizeBytes / (1024 * 1024);
  
  if (sizeMB > maxSizeMB) {
    throw new Error(`File too large: ${sizeMB.toFixed(2)}MB (max ${maxSizeMB}MB)`);
  }
}
```

**Защищает от:**
- Загрузки слишком больших файлов
- Исчерпания памяти сервера
- DoS через большие файлы

---

### 6. Таймауты для fetch

```typescript
export async function fetchWithTimeout(
  url: string,
  options: RequestInit = {},
  timeoutMs: number = 30000
): Promise<Response> {
  const controller = new AbortController();
  const timeout = setTimeout(() => controller.abort(), timeoutMs);
  
  try {
    return await fetch(url, { ...options, signal: controller.signal });
  } finally {
    clearTimeout(timeout);
  }
}
```

**Защищает от:**
- Зависания на медленных запросах
- Исчерпания ресурсов
- DoS через долгие запросы
