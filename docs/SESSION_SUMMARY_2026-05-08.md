## 💡 Ключевые решения

### 1. Singleton pattern для кэшей
**Почему:** Гарантирует единственный экземпляр кэша в памяти

### 2. Эвристический POS tagging
**Почему:** Не требует внешних библиотек, достаточно точен для игровых текстов

### 3. Word alignment с фильтрацией
**Почему:** Улучшает качество извлечения слов из переводов

### 4. 6-уровневый pipeline
**Почему:** Максимальное переиспользование, минимум API вызовов

### 5. Автоматический fallback
**Почему:** Надёжность - система всегда вернёт перевод
