# Инструкции для агентов

Эта папка содержит результаты работы агентов по исследованию и разработке системы кэширования.

---

## 📋 Формат отчетов агентов

Каждый отчет агента должен быть сохранен в отдельный файл с именем:

```
YYYY-MM-DD_HH-MM_<agent-type>_<task-name>.md
```

**Пример:**
```
2026-05-08_17-40_explore_fragment-cache-usage.md
2026-05-08_17-45_plan_ngram-fragmentation.md
```

---

## 🤖 Типы агентов

### Explore (исследование)
- Поиск файлов
- Grep по ключевым словам
- Анализ использования функций

### Plan (планирование)
- Проектирование новых функций
- Архитектурные решения
- Пошаговые планы реализации

### General-purpose (универсальный)
- Сложные многошаговые задачи
- Комплексный анализ
