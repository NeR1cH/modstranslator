# Отчет о тестировании FragmentCache на реальных данных
**Дата:** 2026-05-09

## Проблемы найдены

### 1. ❌ Неправильный род существительного "frame" в словаре
**Проблема:** `'frame': 'masculine'` в NOUN_GENDERS, но "рама" - женский род

**Пример ошибки:**
- "Advanced Iron Frame" → "Продвинутая Железный рама" ❌
- Должно быть: "Продвинутая железная рама" ✅

**Причина:**
- Система определяет род существительного из словаря NOUN_GENDERS
- "frame" был указан как masculine, поэтому прилагательные согласовывались неправильно

**Исправление:**
- Перенесен 'frame' из masculine в feminine (lib/fragmentCache.ts:82)
- Теперь: `'frame': 'feminine'`

**Результат после исправления:**
- "Advanced Iron Frame" → "Продвинутая железная рама" ✅
- "Reinforced Steel Casing" → "Усиленный стальный корпус" ✅

---

### 2. ✅ Регистр букв в составных переводах
**Проблема:** "Продвинутая Железная рама" (с заглавной "Ж")

**Причина:**
- Фрагменты сохраняются с заглавной буквы из оригинального перевода
- При составлении word-by-word все слова вставлялись как есть

**Исправление:**
- Добавлена нормализация регистра: все слова кроме первого - с маленькой буквы
- lib/fragmentCache.ts:432-434

**Результат:**
- "Advanced Iron Frame" → "Продвинутая железная рама" ✅

## Что НЕ работает (ограничения системы)

### ⚠️ Неизвестные существительные
**Проблема:** Если существительное НЕ в словаре NOUN_GENDERS, система не может определить род

**Примеры из теста:**
- "Titanium Wrench" - "wrench" не в словаре
- "Aluminum Cable" - "cable" не в словаре  
- "Platinum Coil" - "coil" не в словаре
- "Silver Wire" - "wire" не в словаре
- "Osmium Dust" - "dust" в словаре (feminine) ✅
- "Uranium Chunk" - "chunk" в словаре (masculine) ✅

**Что происходит:**
- Если существительное неизвестно, `nounGender = null`
- Прилагательные НЕ согласуются, используется форма из кэша как есть
- Может быть правильно (если DeepL дал правильную форму), может быть неправильно

**Решение:**
- Добавить больше существительных в NOUN_GENDERS
- Или: определять род автоматически из окончания русского перевода

## Рекомендации

### 1. Расширить словарь NOUN_GENDERS
Добавить часто встречающиеся существительные из модов:
- wrench (гаечный ключ) - masculine
- cable (кабель) - masculine
- coil (катушка) - feminine
- wire (проволока) - feminine
- casing (корпус) - masculine
- frame (рама) - feminine ✅ (уже исправлено)

### 2. Автоматическое определение рода
Можно определять род из русского перевода по окончанию:
- Окончание на согласную → masculine (кабель, корпус)
- Окончание на -а/-я → feminine (рама, катушка, проволока)
- Окончание на -о/-е → neuter (окно)

### 3. Проверить другие существительные в словаре
Возможно есть еще ошибки в NOUN_GENDERS, как с "frame"
