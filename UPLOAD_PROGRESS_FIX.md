# Исправление визуализации прогресса загрузки файлов

## Дата: 2026-05-03

## Проблема
Визуализация прогресса загрузки файлов была недостаточно информативной:
- Не показывался процент загрузки
- Не было прогресс-бара при загрузке
- Не отображалось количество обработанных файлов

## Решение

### 1. Добавлен процент загрузки (`uploadPercent`)
- Новое состояние для отслеживания процента загрузки
- Обновляется после обработки каждого файла

### 2. Улучшен индикатор загрузки
- Добавлен прогресс-бар с процентами
- Показывается текущий файл и его номер: `[1/3] filename.jar`
- Визуально отделен от остального интерфейса

### 3. Обновлен компонент ProgressBar
- Добавлен опциональный параметр `label`
- Позволяет различать загрузку (`LOADING`) и перевод (`TRANSLATION`)

## Изменённые файлы
- `app/page.tsx` - добавлен `uploadPercent`, улучшена логика отображения прогресса
- `components/ProgressBar.tsx` - добавлен параметр `label`

## Результат
Теперь пользователь видит:
- Точный процент загрузки файлов
- Какой файл обрабатывается (номер и имя)
- Визуальный прогресс-бар с анимацией
- Разделение между загрузкой и переводом
