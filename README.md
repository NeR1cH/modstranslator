# 🟩 MOD_TRANSLATOR — Minecraft Localization Engine

> Автоматический переводчик модов и модпаков Minecraft с английского на русский язык через DeepL API.  
> Работает прямо в браузере — просто перетащи файл, нажми кнопку, скачай готовый результат.

## 📁 Поддерживаемые форматы

| Формат | Версия MC | Описание |
|--------|-----------|----------|
| `.jar` | 1.13+ | Современные моды — ищет `assets/*/lang/en_us.json`, добавляет `ru_ru.json` |
| `.jar` | до 1.13 | Старые моды — ищет `assets/*/lang/en_US.lang`, добавляет `ru_ru.lang` |
| `.zip` | любая | Модпаки — сканирует все файлы внутри, переводит поддерживаемые форматы |
| `.json` | любая | Плоский lang-файл ИЛИ вложенный JSON (квесты, Patchouli-книги) |
| `.lang` | любая | Старый формат языковых файлов (`key=value`) |
| `.snbt` | любая | Квесты FTB Quests / Better Questing |

### Что пропускается автоматически

При обработке модпаков система **пропускает**:
- Бинарные файлы (`.class`, `.png`, `.jpg`, `.ogg`, `.mp3`, ...)
- Вложенные ZIP архивы (`.zip` внутри ZIP)
- Системные папки (`META-INF`, `node_modules`, `.git`)
- Файлы без переводимого текста (менее 2 символов, без латиницы)

**Важно:** JAR файлы модов **НЕ пропускаются** — они обрабатываются автоматически!

## 🚀 Быстрый старт

### 1. Установи Node.js

Скачай с [nodejs.org](https://nodejs.org) — **версия 18 или выше**.

```bash
node --version   # должно быть v18.x.x или выше
npm --version
```

### 2. Получи DeepL API ключ

Зарегистрируйся на [deepl.com/pro-api](https://www.deepl.com/pro-api):

- **Бесплатный план** — 500 000 символов/месяц. Ключ оканчивается на `:fx`
- **Pro план** — без ограничений по символам

### 3. Установи зависимости и настрой окружение

```bash
# Перейди в папку проекта
cd modstranslator

# Установи зависимости
npm install

# Создай файл с ключом из примера
cp .env.example .env       # Mac / Linux
copy .env.example .env     # Windows
```

Открой `.env` и вставь свой ключ:

```env
DEEPL_API_KEY=твой_ключ_deepl_здесь
```

### 4. Запусти

**Универсальное меню (рекомендуется):**
```bash
scripts\dev-menu.bat
```

Меню предлагает:
1. Запустить только тесты
2. Запустить dev-сервер (без тестов)
3. Запустить с проверкой тестов
4. Запустить с авто-исправлением (3 попытки)
5. Выход

**Или используй отдельные команды:**
```bash
npm run dev              # Обычный запуск
npm run dev:safe         # С проверкой тестов
scripts\run-tests.bat    # Только тесты
```

Открой браузер: **http://localhost:3000**

## 🎮 Как пользоваться

### Перевод мода (`.jar`)

1. Перетащи `.jar` файл в зону загрузки (или кликни для выбора)
2. Система проанализирует JAR: покажет количество языковых файлов и строк
3. Нажми **ЗАПУСТИТЬ ПЕРЕВОД** — прогресс отображается в реальном времени
4. Посмотри **ОТЧЕТ О ПЕРЕВОДЕ** — увидишь примеры что было переведено
5. Нажми **СКАЧАТЬ** — получишь исходный JAR с добавленными `ru_ru.json` / `ru_ru.lang`

> Оригинальные английские файлы остаются нетронутыми — добавляются только русские

### Перевод модпака (`.zip`)

1. Перетащи `.zip` архив модпака
2. Система просканирует всё содержимое:
   - **JAR файлы модов** — извлекает и переводит lang файлы внутри каждого мода
   - Языковые файлы (если лежат отдельно)
   - SNBT-квесты FTB / Better Questing
   - Patchouli-книги и квесты (вложенный JSON)
   - Конфиги (TOML, CFG, XML)
3. Нажми **ЗАПУСТИТЬ ПЕРЕВОД**
4. Скачай готовый архив — структура оригинального модпака сохранена

**Пример:** Модпак с 268 JAR файлами модов будет полностью переведён — каждый мод получит свой `ru_ru.json` файл внутри JAR.

### Кэширование переводов

Система автоматически сохраняет все переводы на диск:

```
💾 КЭШ ПЕРЕВОДОВ
Закэшировано: 1,234 запись(ей)
```

**Преимущества:**
- ✅ Повторные переводы используют кэш вместо API
- ✅ Экономия до 70% API лимита
- ✅ Кэш не теряется при перезапуске сервера
- ✅ Работает автоматически, настройка не требуется

**Кэш фрагментов:**
Система умеет переиспользовать части переводов:
```
"Diamond Sword" → "Алмазный меч" (первый раз через API)
"Diamond Axe" → "Алмазный топор" (слово "Diamond" из кэша!)
```

## 🔬 Архитектура

### Поток данных

```
Пользователь загружает файл (base64)
        ↓
POST /api/analyze  →  статистика (строк, файлов)
        ↓
POST /api/translate-stream  →  перевод с прогрессом
        ↓
  ┌─────────────────────────────────────────┐
  │  Роутер по расширению                   │
  │  .zip → translateModpack()              │
  │  .jar → extractLangFiles()              │
  │         translateLangFiles()            │
  │         repackJar()                     │
  │  прочие → getStandaloneHandler()        │
  └─────────────────────────────────────────┘
        ↓
  translateTexts() — проверка кэша
        ↓
  fragmentCache.tryTranslate() — умное переиспользование
        ↓
  DeepL API (только для новых строк)
        ↓
  Сохранение в кэш
        ↓
  Генерация отчета
        ↓
Пользователь скачивает результат
```

### Система кэширования

**Двухуровневый кэш:**

1. **Полный кэш** (`.translation-cache/cache-v1.json`)
   - Сохраняет точные совпадения фраз
   - Используется первым

2. **Кэш фрагментов** (`.translation-cache/fragments-v1.json`)
   - Сохраняет части переводов (материалы, типы предметов)
   - Комбинирует фрагменты для новых переводов
   - Экономит API при переводе похожих модов

**Пример работы:**
```
Мод 1: "Diamond Sword" → API → "Алмазный меч"
       Сохранено: "Diamond" → "Алмазный", "Sword" → "меч"

Мод 2: "Diamond Axe" → 
       "Diamond" найден в кэше (90% уверенность)
       "Axe" не найден → API только "Axe" → "топор"
       Результат: "Алмазный топор"
       Экономия: 50% символов!
```

## 🔐 Безопасность

### Версия 3.10.1 (Security Patch)

Проект защищен от основных угроз:

| Защита | Описание |
|--------|----------|
| 🛡️ **CSRF защита** | Проверка Origin для всех POST/PUT/DELETE запросов |
| ⏱️ **Rate Limiting** | Максимум 20 запросов в минуту на IP |
| 📏 **Валидация размера** | Максимум 800 MB на файл (ограничение памяти браузера) |
| 🚫 **Path Traversal защита** | Невозможно перезаписать системные файлы через `../../` |
| 🔒 **Prototype Pollution защита** | Безопасный парсинг JSON |
| ⏰ **Таймауты** | 30 секунд на каждый запрос к DeepL API |
| 🔑 **Безопасность ключа** | API ключ хранится только на сервере |

**Файлы безопасности:**
- `lib/security.ts` — утилиты безопасности
- `middleware.ts` — CSRF и rate limiting

## 🛠️ Устранение неполадок

### ❌ `DEEPL_API_KEY не задан`
- Проверь наличие файла `.env` в корне проекта
- Убедись, что ключ вставлен без лишних пробелов и кавычек
- Перезапусти сервер после изменения `.env` (`Ctrl+C` → `npm run dev`)

### ❌ `Нет английских lang файлов в JAR`
- Мод может не содержать локализацию вовсе
- Языковые файлы могут быть в нестандартной папке
- Текст мода может быть встроен напрямую в код

### ❌ Ошибка 429 (Too Many Requests)
- Превышен лимит запросов к DeepL API
- Система автоматически повторит попытку до 3 раз
- Проверь остаток символов на [deepl.com/account/usage](https://www.deepl.com/account/usage)

### ❌ Ошибка 429 от сервера (Rate Limit)
- Превышен лимит 20 запросов в минуту
- Подожди 1 минуту и попробуй снова

### ❌ `File too large (max 1.5GB)`
- Размер файла превышает 1.5 GB
- Попробуй разбить модпак на части или удалить самые большие моды

### ❌ Файл не загружается
- Проверь размер файла (максимум 1.5 GB)
- Для файлов больше 800 MB используется streaming upload (может занять больше времени)
- Убедись, что расширение поддерживается
- Открой консоль браузера (F12 → Console) для деталей
